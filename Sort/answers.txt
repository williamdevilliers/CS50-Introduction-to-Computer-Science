sort1 uses: Bubble sort

How do you know?: Bubble sort is notorious for sorting reverse order lists slowly and famous for sorting sorted lists fast.

sort2 uses: Merge sort

How do you know?: Merge sort is by far the fastest sorting method of the 3. It scores the fastest time for all lists.

sort3 uses: Selection sort

How do you know?: Selection sort takes roughly the same amount of time to sort any similar sized list. A property of slection sort is that it needs to scan through each list as many times as there are elements in said list. This is why the sorting times are similar for each list of the same size.
